## Getting Started

If you're new to the EMS Dashboard, we recommend following this order:

1. **Configure Agency** - Set up your agency information and settings
2. **Add Units** - Register your EMS units and vehicles
3. **Add Users** - Set up team members and assign roles
4. **Review Available Trips** - Learn how to accept trip requests
5. **Manage My Trips** - Track and update trip status

## Common Tasks

### Accepting a Trip
1. Go to **Available Trips** tab
2. Review trip details
3. Check distance and requirements
4. Click **Accept** button
5. Assign a unit (if needed)

### Updating Trip Status
1. Go to **My Trips** tab
2. Find the trip you want to update
3. Click status update button (e.g., "Mark In Progress")
4. Confirm status change

### Managing Units
1. Go to **Units** tab
2. Add new unit or edit existing
3. Set unit type and capabilities
4. Assign unit to trips as needed
