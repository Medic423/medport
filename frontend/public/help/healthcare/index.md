## Getting Started

If you're new to the Healthcare Dashboard, we recommend following this order:

1. **Configure Settings** - Set up your facility information and preferences
2. **Add Destinations** - Save frequently used locations
3. **Add EMS Providers** - Build your preferred provider list
4. **Create Your First Request** - Follow the Create Request guide
5. **Manage Requests** - Learn to dispatch and monitor trips

## Common Tasks

### Creating a Transport Request
1. Go to **Create Request** tab
2. Fill in patient information
3. Select transport details
4. Choose locations
5. Set ready window
6. Submit request

### Dispatching a Trip
1. Go to **Transport Requests** tab
2. Find the trip you want to dispatch
3. Click **Dispatch** button
4. Select EMS agencies
5. Confirm dispatch

### Managing Settings
1. Go to **Hospital Settings** tab
2. Update facility information
3. Manage locations (if multi-location enabled)
4. Configure notification preferences
