### Hospital Settings

#### Configure Facility Settings & Preferences
The data you enter into these menu items configure **Tracc** to the unique needs of your facility. The **Hospital Settings** tab and its sub menu's **System Settings** and **Manage Locations** <u>must</u> be configured during initial setup.  These values are passing into drop-down menus like **Create Requests**. You can and also set default values for each of the categories.  You can return to **Hospital Settings** anytime you need to add, update or delete facility information.

#### The System Settings Sub Tabs
There are four sub-tabs in **System Settings:** Dropdown Options, Pickup Locations and Main Contact. Tap a category to manage its dropdown options.

### Category Options
These options are the main driver of the trip creation process. You have seven category options You <u>can't</u> add categories but you can change the names. The default names are just suggestions. You should work through the trip creation process to work out exactly what categories of information you need.  

### Add Functionality To Create / Edit Categories

Select a category...
Transport Levels
Urgency Levels
Primary Diagnosis
Mobility Levels
Insurance Companies Secondary Insurance

For example if we choose the **Transport Levels** category you'll be give these options:
- Add New Transport Levels
- Default Transport Levels
- Current Transport Levels

• Add new options using the form above 
• Edit existing options by clicking the "Edit" button 
• Delete options by clicking the "Delete" button 
• Changes will be reflected in the transport request forms immediately

The same procedure is used for all **Categories** that are in the **Select Category** dropdown.

### Pickup Locations 
Pickup locations are the floor, units or departments in a facility where EMS or other transport agency routinely go to pickup patients for transport.
 
#### Instructions
- Select a hospital to manage its pickup locations
- Add new pickup locations using the "Add Pickup Location" button 
- Edit existing locations by clicking the "Edit" button 
- Delete locations by clicking the "Delete" button 

Pickup locations will be available in the trip creation form

###  Main Contact
#### Instructions
- This contact person will be the main administrator for this facility's TCC application
- They will receive notifications about system updates and important changes
- This information may be used for support and troubleshooting purposes
- Ensure the email address is monitored regularly for important communications

### Manage Locations
Manage the hospitals, clinics, and facilities under your organization
#### How to use
• Add all hospitals, clinics, and facilities your organization manages
• The first location you add will automatically be set as primary
• When creating transport requests, you'll select which location needs transport
• You can set a different location as primary at any time by clicking the star icon
• Edit or delete locations as needed (cannot delete if trips exist)
