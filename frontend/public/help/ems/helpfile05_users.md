### Users 

#### Overview
- Purpose: Manage users for your EMS agency
- When to use: To add/remove team members, manage permissions

#### Key Features
- **User List**
  - All users for your agency
  - User roles
  - Active status
- **Add User**
  - Email address
  - Name
  - Role assignment
- **Edit User**
  - Update information
  - Change permissions
  - Reset password
- **Remove User**
  - Deactivate user

#### User Roles
- Org Admin (if multi-agency)
- Standard User
- Dispatcher (if role exists)
