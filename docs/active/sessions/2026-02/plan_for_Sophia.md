## How to Export from Figma

### Exporting Colors
1. Select any element with the color you want
2. In the right panel, click on the color fill/stroke
3. Copy the hex code (e.g., #2563EB)
4. Or use Figma's "Copy as CSS" feature

### Exporting Typography
1. Select text element
2. In the right panel, you'll see:
   - Font family
   - Font size
   - Font weight
   - Line height
   - Letter spacing
3. Note these values down

### Exporting Images
1. Select the image/frame you want to export
2. In the right panel, click the "Export" section
3. Choose format: PNG (for images) or SVG (for icons/illustrations)
4. Choose size: 2x (for retina displays)
5. Click "Export [filename]"

### Exporting Screenshots
1. Select the entire frame/page
2. Use Figma's "Export" feature, or
3. Use browser screenshot tools, or
4. Use Figma's "Share" â†’ "Copy link" and we can view it
